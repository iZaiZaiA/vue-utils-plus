# 其它方法函数

其它有些杂项的方法函数

```javascript
import { utilsOther } from "vue-utils-plus"
```

## 控制台打印

`clog()` `支持版本：V1.0.4`

### 参数说明

| 参数    | 类型  | 可选值 | 默认值  | 说明        |
|-------|-----|-----|------|-----------|
| name  | -   | -   | -    | 带颜色的名称    |
| tips  | -   | -   | -    | 带颜色的字符    |
| data  | -   | -   | -    | 无颜色的数据    |
| micro | -   | -   | '打印' | 第一个带颜色的名称 |

### code示例

```javascript
const { clog } = utilsOther()

clog('通常是模块名', '通常是字段名', 'data数据', 'vup' )
```

| name | -   | -   | -   | 带颜色的名称 |
