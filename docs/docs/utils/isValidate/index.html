isAlphabets() 支持版本：V0.0.2

# 参数说明

参数类型可选值默认值说明

类型说明

formValidate() 支持版本：V0.0.7

# code示例

const { formValidate } = isValidate()

const formRef = ref(null)

const onSubmit = async () => {
    //注意，此方法是异步的。
   const res = await formValidate(formRef.value)
    console.log(res) // true or false
}
