# 是否为某个类型

is() 支持版本：V0.0.2

# 参数说明

参数类型可选值默认值说明
val---判断的内容，如 {}
typestring--类型名称，如 'Object'

# 返回内容

类型说明
Boolean和传递的类型一致，就返回 true，否则返回 false

# code示例

const { is } = isType()

is({}, 'Object')

# 取数组数据

getArrValue() 支持版本：V0.0.6

# 参数说明

参数类型可选值默认值说明
val---判断的内容

# code示例

const { getArrValue } = isType()
// 比如，后端接口返回的一个数据。
// 但我们无法预知，我们要取的data是否为数组
// 有可能在某些情况下，不是数组。
const res = {}
// 如果数据存在，就返回原始数据 res.data.data
// 如果不存在，就返回 空数组 []
getArrValue(res.data.data)
