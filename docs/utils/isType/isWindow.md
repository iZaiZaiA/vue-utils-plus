# `isWindow` 是否为浏览器

`支持版本：V0.0.2`


## 返回内容 {#return}

| 类型      | 说明                       |
|---------|--------------------------|
| Boolean | 是就返回 `true`，否则返回 `false` |


## code示例 {#code}

```javascript
import { isType } from "vue-utils-plus"
const { isWindow } = isType()

console.log(isWindow())
```
