# `getRandom` 生成随机字符串

`支持版本：V0.0.3`


## 参数说明 {#parameter}

| 参数  | 类型  | 可选值 | 默认值 | 说明  |
|-----|-----|-----|-----|-----|
| num | -   | -   | 8   | 长度  |


## 返回内容 {#return}

| 类型  | 说明       |
|-----|----------|
| -   | 返回生成后的数据 |


## code示例 {#code}

```javascript
import { utilsRandom } from "vue-utils-plus"
const { getRandom } = utilsRandom()

console.log(getRandom())
```
